# ⌨ Typing Speed Game

An interactive typing speed test application to measure accuracy and words per minute (WPM).

---

## 🚀 Project Overview
This application tracks typing performance using a timer and dynamically generated text. It calculates typing speed and accuracy in real time.

---

## ✨ Features
- ⏱ Countdown timer
- 📊 WPM calculation
- 🎯 Accuracy tracking
- 🔄 Dynamic text generation

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Date API, Timing Functions, Event Handling)
